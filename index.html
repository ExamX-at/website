Partners

Cheating is not possible, and our software is easy to use.

Provisioned VMs

We provide a website where virtual machines can be requested. Students connect to the VM and there the testing takes place.

Kiosk Mode

Our software runs in kiosk mode, so a VM can only be accessed through it. Thus, students cannot use a file explorer, depriving them of a possibility to cheat.

Monitoring

A simple monitoring program runs in the background so that cheating attempts can be detected quickly and countermeasures can be taken.

Current Situation

Currently it is difficult to perform an audit, because to do it in a secure environment you need a room full of PCs running the latest updates. Doing an audit on a notebook you brought with you was not a good idea so far, because you can always open a file explorer and thus cheat.

Our Solution

We offer a solution to said problem. VM templates are provided for every possible type of testing. An examiner can now request as many VMs as he has examinees on our website. In the background clones are created from the template. The examinees boot into the kiosk mode during this time. After that IP addresses to the VMs are provided. Now you can connect to the VM via the IP and the test can begin.

Monitoring

We have built an efficient monitoring to ensure that examx are conducted in a fair and compliant manner. The monitoring also ensures that no network connections can be established between the individual VMs. Furthermore, the monitoring provides reporting and alerting so that an admin can be notified of certain events.
